@Gridpak

The gridulator allows you to set and export a fluid or adaptive grid at various layouts into one neat little package containing a CSS file with the appropriate media queries and grid, PNGs to use in your graphics program and a Javascript file that allows you to toggle the grid on and off in any browser.
